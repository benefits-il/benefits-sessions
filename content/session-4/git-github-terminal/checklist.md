# צ'קליסט מוכנות לפני השיעור
עברו על הרשימה הזו לפני שאתם מגיעים לשיעור. אם משהו לא מסומן, שלחו צילום מסך בקבוצת ה-WhatsApp כדי שנתקן את זה לפני שנתחיל — הגעה כשכל הדברים עובדים שומרת על קצב ההתקדמות של הכיתה כולה.

- [ ] צפיתי בסרטון הבית ועשיתי את ה-follow-along.
- [ ] Git מותקן (הרצת `git --version` מחזירה מספר גרסה).
- [ ] יש לי תיקיית my-project עם commit אחד לפחות.
- [ ] פתחתי חשבון GitHub ואימתתי את כתובת האימייל.
- [ ] שלחתי צילום מסך ב-WhatsApp אם משהו לא עבד.
