## בבית: להשלים את ארגז הכלים של Benefit

בכיתה בניתם את Benefit, נתתם לו את שני הסקילים הראשונים, שמירה ושליפה, והאכלתם אותו בכל מה שלמדנו עד עכשיו. עכשיו, בבית, משלימים את ארגז הכלים: שני הסקילים שנשארו, STUDIO ו-MICRO. מתקינים כל אחד, ומריצים אותו פעם אחת על חומר אמיתי שלכם.

ואם לא הספקתם לסיים את Benefit בכיתה, כאן המקום להשלים: פותחים פרויקט חדש בקלוד, מדביקים את הסוכן המוכן להוראות הפרויקט, מחברים את Notion עם הרשאת קריאה וכתיבה, ומעלים את קבצי הידע. הכל בעמוד אחד: [benefits-il.dev/benefit-agent](https://benefits-il.dev/benefit-agent).

## הסקיל הראשון — ייעוץ ופירוק לפי STUDIO

לוקח משימה אמיתית שלכם ומפרק אותה לשש תחנות, עם המלצה באיזה כלי להשתמש בכל שלב. מתקינים אותו מעמוד ההורדה דרך Settings ← Skills ← Upload, ומריצים פעם אחת:

```
יש לי משימה: [המשימה שלכם בשורה אחת]. תפרק לי אותה לפי STUDIO, ותגיד באיזה כלי כדאי להשתמש בכל שלב.
```

## הסקיל השני — בונה פרומפט לפי MICRO

בונה לכם פרומפט נקי ומסודר, ומחזיר אותו מוכן להעתקה. מתקינים אותו מאותו עמוד, ומריצים פעם אחת:

```
אני צריך פרומפט ש[מה שאתם רוצים שהפרומפט יעשה]. תבנה לי אותו לפי MICRO.
```

**למה זה:** כל סקיל הוא עוד דבר אחד ש-Benefit עושה בשבילכם. ואתם סומכים על סקיל רק אחרי שראיתם אותו עובד על החומר שלכם, לא לפני. הרצה אחת זה כל מה שצריך.

## כמה דברים שכדאי לזכור

הזיכרון תמיד בידיים שלכם: אפשר לפתוח את Notion ולתקן שם או תאריך בעצמכם. עדיף לתת ל-Benefit לתייק, אבל כשצריך, אתם יכולים. אם פעולת כתיבה לא עוברת, ודאו שחיבור Notion מחובר עם הרשאת כתיבה, ואשרו את כלי הכתיבה כשהוא מבקש. ואם נתקעתם, נסו שוב, ונסדר את זה במפגש.
