## ✏️ בכיתה: בונים את Benefit, ונותנים לו את שכבות הזיכרון

הגיע הרגע להקים את Benefit, הסוכן שיושב לידכם ומחזיק את הזיכרון. בכיתה אנחנו עושים את הליבה: מקימים אותו, נותנים לו את שני הסקילים הראשונים, שמירה ושליפה, מאכילים אותו בכל מה שלמדנו עד עכשיו, ואז שולפים את זה בחזרה במו עינינו. את שני הסקילים הנוספים, ואת השאר, נשלים בבית.

## צעד 1 — מקימים את Benefit

פותחים פרויקט חדש בקלוד בשם Benefit, ומדביקים את הסוכן המוכן לתוך הוראות הפרויקט (Instructions). מחברים את Notion עם הרשאת קריאה וגם כתיבה, ומעלים ל-Files את כל קבצי הידע, וביניהם קובץ מבנה-הקורס. הכל יושב בעמוד אחד: [benefits-il.dev/benefit-agent](https://benefits-il.dev/benefit-agent).

## צעד 2 — מתקינים את שני הסקילים הראשונים: שמירה ושליפה

מאותו עמוד מורידים שני סקילים ומתקינים אותם בקלוד דרך Settings ← Skills ← Upload, כל אחד בנפרד:
- **שמירה** — Benefit כותב לכם תובנה או עמוד חדש לתוך Notion.
- **שליפה** — Benefit ניגש לאינדקס, בוחר את העמוד הרלוונטי, וטוען רק אותו (index ← select ← load).

את שני הסקילים הנוספים, STUDIO ו-MICRO, תתקינו בבית.

## צעד 3 — מאכילים את Benefit בכל מה שלמדנו עד עכשיו

עכשיו נותנים ל-Benefit את כל מה שלמדנו, כדי שתהיה לו התמונה המלאה. שמונה פרקי-למידה, משני השיעורים: משיעור 1, יסודות, סקילים, סוכנים. ומההכנה לשיעור 2, שפה משותפת, connectors, נושן, שרשור סוכנים, ו-deep research.

פותחים שיחה חדשה עם Benefit. נכנסים לעמוד של כל פרק-למידה באתר, לוחצים על הכפתור "העתק לנושן", ומדביקים את הפרק לשיחה. בראש השיחה, לפני הפרק הראשון, מדביקים את הפרומפט הבא:

```
אני אדביק לך עכשיו פרקי-למידה מהקורס, אחד-אחד. לכל פרק: זהה לאיזו יחידה ולאיזה שיעור הוא שייך, מצא לו את המקום הנכון במחברת (Benefits ← השיעור ← היחידה), והוסף לו את תיוג ה-STUDIO המתאים. לפני שאתה כותב משהו לנושן, הראה לי קודם איפה הפרק הולך ואיזה תיוג נתת, וחכה לאישור שלי. מתחילים. הפרק הראשון:
```

מתחת לפרומפט מדביקים את הפרק הראשון. אחרי ש-Benefit הראה לכם איפה הוא מתייק ואישרתם, כותבים "הפרק הבא:" ומדביקים את הבא, וכך עד ששמונת הפרקים בפנים.

**למה זה:** זה מה שהופך את Benefit מ"מכיר את השיעור של היום" ל"מכיר את כל הקורס". והתיוק לפי המבנה הוא בדיוק מה שיאפשר לו אחר כך לשלוף לכם את העמוד הנכון בלי לחפש בעיניים עצומות.

## צעד 4 — שולפים מידע קודם מהקורס

עכשיו בודקים שהשליפה עובדת. פותחים שיחה חדשה עם Benefit, ושואלים שאלת תוכן על משהו שכבר למדתם. Benefit ניגש לאינדקס ב-Notion, בוחר את העמוד הרלוונטי, טוען רק אותו, ועונה, עם הפנייה לאיפה זה שמור. הדביקו:

```
תזכיר לי מה זה שיטת MICRO, ומאיפה במחברת שלי שלפת את זה. תסביר בקצרה, ותראה לי את העמוד שעליו הסתמכת.
```

**למה זה:** ככה אתם רואים במו עיניכם שהזיכרון לא רק נכנס, אלא גם יוצא בחזרה לפי-צורך. זה כל ההבדל בין ערימת מסמכים לבין סוכן שזוכר.

## כמה דברים שכדאי לזכור

- הזיכרון תמיד בידיים שלכם: אפשר לפתוח את Notion ולתקן שם או תאריך בעצמכם. עדיף לתת ל-Benefit לתייק, אבל כשצריך, אתם יכולים.
- אם פעולת כתיבה לא עוברת, ודאו שחיבור Notion מחובר עם הרשאת כתיבה, ואשרו את כלי הכתיבה כשהוא מבקש.
- אם נתקעתם באיזשהו שלב, נסו שוב, ונסדר את זה יחד במפגש.
