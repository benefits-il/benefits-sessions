# בבית — בניית Plugin

## קודם כול, מסיימים את מה שהתחלתם

בכיתה התחלתם לבנות plugin, ב-Linear או ב-Canva. הדבר הראשון בבית הוא לסיים אותו. פתחו את ה-Plugin Create מאיפה שעצרתם, השלימו את השאלות שנשארו פתוחות, התקינו את ה-plugin ב-Cowork, והריצו אותו פעם אחת על משהו אמיתי. לא הדגמה, משהו שלכם: ערימת המשימות שלכם ללוח, או עובדות על המוצר שלכם לעמוד מכירה. כשאתם רואים את התוצר יוצא, סיימתם את החלק הראשון.

## ועכשיו, בונים עוד — הפעם לעולם שלכם

בכיתה נשארנו בעולם של Tuesday. בבית עוברים לעולם שלכם. למטה מחכה מאגר של עשרים פרויקטים, מחולקים לארבע קבוצות. כל אחד הוא plugin שלם עם פרומפט-פתיחה מוכן: מעתיקים אותו ל-Plugin Create, עונים על השאלות, ומקבלים כלי.

אל תבנו את כולם. בנו אחד או שניים שבאמת יעזרו לכם, מהעולם האמיתי שלכם, העבודה או הבית. אחרי המאגר יש טבלה שתעזור לכם לבחור חכם, כדי שכמה בניות יחד יגעו בכל סוגי היכולות.

## המאגר — עשרים פרויקטים

### קבוצה א · כיף ואישי

**1. מתכנן הטיול המושלם** — מקבל יעד, תאריכים ותקציב, ובונה מסלול יומי עם טיסות, מלונות והמלצות.

```
אני רוצה לבנות plugin שמתכנן טיולים. הוא מקבל ממני יעד, תאריכים ותקציב, מחפש טיסות ומלונות דרך Expedia ו-Booking, מצליב המלצות מ-Tripadvisor, ומפיק מסמך עם מסלול יומי והערכת עלות. תעזור לי לבנות אותו צעד אחר צעד.
```

**2. אוצר המוזיקה / DJ אישי** — מתאר לו מצב-רוח או אירוע, והוא בונה playlist ב-Spotify.

```
אני רוצה לבנות plugin שהוא DJ אישי. אני מתאר לו מצב-רוח או אירוע, והוא בונה לי playlist ב-Spotify, מסביר למה בחר כל שיר, ויכול גם לנקות לי את הספרייה. תעזור לי לבנות אותו.
```

**3. מאתר הופעות ואירועים** — מוצא הופעות לפי הטעם שלכם, בודק כרטיסים, ומסנכרן ליומן.

```
אני רוצה plugin שמוצא לי הופעות ואירועים לפי הטעם שלי והאזור שלי, בודק כרטיסים ב-StubHub, ומוסיף את מה שמעניין אותי ליומן Google. תעזור לי לבנות אותו.
```

**4. מתכנן האירוע** — מתכנן מסיבה מקצה לקצה: רשימת אורחים, לוח זמנים, קניות והזמנות.

```
אני רוצה plugin שמתכנן אירועים. אני אומר לו איזה אירוע ולכמה אנשים, והוא בונה צ'קליסט, לוח זמנים, רשימת קניות, וטיוטות הזמנה במייל. תעזור לי לבנות אותו.
```

**5. השף הביתי** — אומרים לו מה יש במקרר וכמה אנשים, והוא בונה תפריט ורשימת קניות.

```
אני רוצה plugin שהוא שף ביתי. אני אומר לו מה יש לי במקרר וכמה אנשים, והוא מציע תפריט ובונה רשימת קניות מסודרת. תעזור לי לבנות אותו.
```

### קבוצה ב · מקצועי ומרשים

**6. מכונת המותג** — ממדריך מותג ותוכן, מפיק פוסטים, באנרים ומצגות ב-Canva.

```
אני רוצה לבנות plugin שהוא מכונת מותג. נותנים לו מדריך מותג ותוכן, והוא מפיק לי ב-Canva פוסטים, באנרים ומצגות בצבעים ובסגנון של המותג שלי. תעזור לי לבנות אותו.
```

**7. גשר העיצוב** — קורא קובץ Figma ומפיק מסמך handoff מסודר למפתח.

```
אני רוצה plugin שקורא קובץ Figma שלי ומפיק מסמך handoff מסודר למפתח, כולל רשימת רכיבים, צבעים ו-design tokens. תעזור לי לבנות אותו.
```

**8. מנתח המכירות** — מתחבר ל-PayPal, מסכם עסקאות, ומזהה מגמות.

```
אני רוצה plugin שמנתח לי את המכירות. הוא מתחבר ל-PayPal, מסכם עסקאות, מזהה מגמות, ומראה לי אילו חשבוניות פתוחות. תעזור לי לבנות אותו.
```

**9. מתאם הפגישות** — מתאם פגישות דרך Calendly, ומכין דף הכנה לכל פגישה.

```
אני רוצה plugin שמתאם לי פגישות דרך Calendly, ולפני כל פגישה מכין לי דף הכנה מהמיילים והיומן שקשורים למשתתפים. תעזור לי לבנות אותו.
```

**10. בנאי עמוד הנחיתה** — בונה עמוד נחיתה ב-Wix מתוך תיאור וטקסט.

```
אני רוצה plugin שבונה לי עמוד נחיתה ב-Wix מתוך תיאור וטקסט שאני נותן לו. תעזור לי לבנות אותו.
```

**11. בונה הלוח החזותי** — לוקח תיקיית הערות ובונה מהן לוח Miro מסודר.

```
אני רוצה plugin שלוקח תיקיית הערות ורעיונות ובונה מהם לוח Miro מסודר עם קבוצות וקשרים. תעזור לי לבנות אותו.
```

### קבוצה ג · נועז ועצמאי

**12. הסוכן שעושה במקומך** — נותנים לו מטרה אחת, והוא מחפש, ממלא טבלה, ומכין טיוטות.

```
אני רוצה plugin שעושה משימות במקומי. אני נותן לו מטרה אחת, למשל "מצא לי את שלושת הספקים הכי טובים לX, השווה אותם, ותכין מייל פנייה לכל אחד". הוא מחפש ברשת, ממלא טבלה, ומכין טיוטות (בלי לשלוח). תעזור לי לבנות אותו.
```

**13. מכונת התוכן** — מווידאו או מסמך, מפיק פוסט, thread, קרוסלה וניוזלטר.

```
אני רוצה plugin שהוא מכונת תוכן. אני נותן לו וידאו, מסמך או תיקיית הערות, והוא מפיק לי פוסט LinkedIn, thread ל-X, טקסט לקרוסלה, וטיוטת ניוזלטר על אותו רעיון. תעזור לי לבנות אותו.
```

**14. סוכן המודיעין התחרותי** — סורק את המתחרים ברשת ומפיק דוח קצר.

```
אני רוצה plugin שעוקב אחרי המתחרים שלי. הוא מחפש ברשת מה השתנה אצלם, מזהה פערים מולי, ומפיק דוח קצר. תעזור לי לבנות אותו.
```

**15. המנתח האישי** — מצביעים על תיקייה, והוא מוצא דפוסים ומפיק דוח תובנות.

```
אני רוצה plugin שמנתח אותי. אני מצביע על תיקייה עם הערות, יומן וייצוא שיחות, והוא מוצא דפוסים ותמות ומפיק לי דוח תובנות אישי. תעזור לי לבנות אותו.
```

### קבוצה ד · יומיומי וקליל (העוגן הבטוח)

**16. עוזר המייל האישי** — עובר על התיבה, מסווג, ומכין טיוטות מענה.

```
אני רוצה plugin שעוזר לי עם המייל. הוא עובר על התיבה, מסווג כל הודעה לדחוף/לעיון/רעש, ומכין טיוטות מענה בסגנון שלי. בלי לשלוח. תעזור לי לבנות אותו.
```

**17. בריף הבוקר** — כל בוקר מסכם את היומן, המיילים החשובים והמשימות הפתוחות, אוטומטית.

```
אני רוצה plugin שמכין לי בריף בוקר. כל בוקר הוא מסכם את היומן, את המיילים החשובים ואת המשימות הפתוחות, ושומר קובץ שמחכה לי. תעזור לי לבנות אותו וגם להגדיר שירוץ אוטומטית כל בוקר.
```

**18. מסכם פגישות הזום** — לוקח תמליל פגישה ומפיק סיכום, action items וטיוטת מעקב.

```
אני רוצה plugin שלוקח תמליל פגישת זום ומפיק לי סיכום, רשימת action items, וטיוטת מייל מעקב למשתתפים. תעזור לי לבנות אותו.
```

**19. המסדר החכם** — קורא את תוכן הקבצים, ממיין לפי נושא, ונותן שמות הגיוניים.

```
אני רוצה plugin שמסדר לי תיקיות. הוא קורא את התוכן של כל קובץ, ממיין לפי נושא, נותן שמות הגיוניים, ומראה לי לוג של מה שינה. תעזור לי לבנות אותו.
```

**20. בונה הסטודיו התלת-ממדי** — מתיאור מילולי בונה סצנה תלת-ממדית אינטראקטיבית בדפדפן.

```
אני רוצה plugin שבונה חוויות תלת-ממד בדפדפן עם Three.js. אני מתאר סצנה או אובייקט, והוא מייצר לי עמוד אינטראקטיבי. תעזור לי לבנות אותו.
```

## איך לבחור חכם — טבלת היכולות

## הכלל הפשוט מאחורי הטבלה

יש שני פרויקטים שהם המשלימים:

- **17, בריף הבוקר** הוא היחיד שרץ בתזמון אוטומטי, וגם מביא חיבור ליומן ולמייל. בלעדיו לא תיגעו אף פעם בהרצה אוטומטית.
- **13, מכונת התוכן** מביא עבודה על קבצים, חיפוש ברשת, וסוכן שמבצע מקצה לקצה, שלוש יכולות במכה אחת.

לכן הכלל פשוט: בנו קודם את הפרויקט שבאמת יעזור לכם, ואז הוסיפו את 17 ואת 13. שלוש בניות, ונגעתם בכל סוג יכולת, חיבור, תזמון, קבצים, רשת, סוכן-מבצע ופלט מקצועי. ואם כבר בניתם את 12 או 13, אתם כמעט שם, חסר לכם רק התזמון של 17.

זה לא תרגיל לשם התרגיל. כל בנייה כזאת היא כלי אמיתי שיישאר אצלכם אחרי השיעור.
