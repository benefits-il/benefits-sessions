# At home — Memory Layers

In class you built Benefit, gave it its first two skills, save and retrieve, and fed it everything we've learned so far. Now, at home, you complete its toolkit: the two skills that are left, STUDIO and MICRO. Install each one, and run it once on real material of your own.

And if you didn't get to finish Benefit in class, here's where you complete it: open a new project in Claude, paste the ready-made agent into the project instructions, connect Notion with read and write permission, and upload the knowledge files. It's all on one page: [benefits-il.dev/benefit-agent](https://benefits-il.dev/benefit-agent).

## The first skill — advising and breaking down by STUDIO

It takes a real task of yours and breaks it into the six stations, with a recommendation of which tool to use at each stage. Install it from the download page through Settings → Skills → Upload, and run it once:

```
I have a task: [your task in one line]. Break it down for me by STUDIO, and tell me which tool to use at each stage.
```

## The second skill — building a prompt by MICRO

It builds you a clean, ordered prompt and hands it back ready to copy. Install it from the same page, and run it once:

```
I need a prompt that [what you want the prompt to do]. Build it for me by MICRO.
```

**Why this:** each skill is one more thing Benefit does for you. And you trust a skill only after you've seen it work on your own material, not before. One run is all it takes.

## A few things worth remembering

The memory is always in your hands: you can open Notion and fix a name or a date yourself. It's better to let Benefit do the filing, but when you need to, you can. If a write doesn't go through, check that Notion is connected with write permission, and approve the write tool when it asks. And if you get stuck, try again, and we'll sort it out together at the session.
