## ✏️ תורידו את הכלי שמקל על ה-AI

ראיתם בסרטון שהדרך הכי פשוטה לחסוך בטוקנים היא להקל על ה-AI. אז לפני שנבין לעומק למה, בואו פשוט נשיג כלי מוכן שעושה את זה. (כן, עוד לפני ההסבר. סומכים עליכם.)

## מה הכלי עושה

לוקח כל קלט מבולגן, קובץ Word, PDF, מצגת, תמונה או סתם טקסט, ומחזיר אותו כ-MD נקי ומסודר. בדיוק "להקל על ה-AI" שדיברנו עליו.

## המשימה

1. היכנסו לסקיל בקישור: [markitdown של Eleanor Berger](https://gist.github.com/intellectronica/7c76818d79851fa8c33a6f7b0420bccb).
2. הורידו מהקישור את קובץ הסקיל (`.skill`), והעלו אותו לקלוד, כמו שהתקנתם סקילים בשיעור הראשון.
3. זהו. מעכשיו יש לכם כלי שמסדר ל-AI את החומר עוד לפני שהוא ניגש לעבוד.

## בדיקה עצמית

- הסקיל מותקן ומופיע ברשימת הסקילים שלכם בקלוד.

## טיפ

הסקיל הזה הגיע מהספרייה החופשית שלנו, [benefits-il.dev/benefits-library](https://benefits-il.dev/benefits-library). יש שם מדף שלם של כלים כאלה, תציצו מתי שמתחשק.

## ✏️ הכלי: לנקות, לייצא, ולשלוח

את הכלי הזה כבר פגשתם בכיתה, אז לא נכיר אותו מחדש. רק נתרגל. [md-he](https://benefits-il.dev/md-he) פתוח בלשונית, בלי התקנה ובלי חשבון.

## המשימה — ארבע פעולות, כל אחת דקה

1. **נקה והדבק.** קחו פלט עברי כלשהו מקלוד, העתיקו, ובכלי לחצו "נקה והדבק". פתאום זה מסודר וקריא, גם מימין.
2. **ייצוא ל-Word.** את אותו טקסט, ייצאו ל-Word, וראו איך הכותרות הפכו לכותרות אמיתיות.
3. **העתקה לוואטסאפ.** קחו רשימה קצרה, בחרו העתקה לוואטסאפ, ושלחו לעצמכם. הבולדים יוצאים נכון.
4. **Mermaid.** הדביקו בכלי את הטקסט הבא וראו איך הוא הופך לתרשים:

```
flowchart TD
  A[פלט מקלוד] --> B[נקה והדבק בכלי]
  B --> C[ייצוא ל-Word או וואטסאפ]
```

## בדיקה עצמית

- הדבקתם רגיל והפורמט נשבר? ודאו שלחצתם "נקה והדבק" ולא הדבקה רגילה.
- וואטסאפ לא מציג כותרות וטבלאות, רק בולד, איטליק ורשימות. זה תקין.

## ✏️ חקרו ← המירו ← זהו

הכרתם את המשפחה בסרטון. עכשיו תפגשו אותה ביד. שלושה שלבים, והבטחה אחת: לא כותבים אף שורה.

## שלב 1 — חקרו: אותו תוכן, ארבע תחפושות

פתחו את קלוד והדביקו את הפרומפט (הוא כבר מכיל פריט לדוגמה, פשוט שלחו):

<!-- verbatim (פרומפט מאושר) — לא לתרגם ולא לקצר -->
```
קח את הפריט הבא והצג אותו בארבעה פורמטים, בזה אחר זה: Markdown, YAML, JSON, ו-XML.
מתחת לכל פורמט כתוב שורה אחת בלבד: מתי הוא הכי שימושי ואיפה נתקלים בו.
בלי הסבר ארוך, רק הדוגמה והשורה.

הנה הפריט:
שם: דנה לוי
תפקיד: מנהלת מוצר
עיר: חיפה
```

אותו תוכן בדיוק, ארבע צורות שונות לגמרי.

## שלב 2 — המירו: תנו לסקיל לעבוד

זוכרים את הסקיל שהתקנתם קודם? תנו לו עבודה אמיתית: קחו קובץ או טקסט מבולגן משלכם מהשבוע, וזרקו אותו לסקיל. הוא יחזיר אותו כ-MD נקי, בלי לשנות מילה. (אין לכם משהו ביד? השתמשו בקטע הזה.)

```
פגישה 26 במאי. השתתפו רונית, דני ומיכל. החלטות: היקף יוגבל לשלושה מודולים, השיטה sprints של שבועיים. נקודה פתוחה: ספק חיצוני לאינטגרציה.
```

## שלב 3 — זהו: שלוש שאלות

לכל קטע, איזה פורמט אתם רואים? בחרו תשובה אחת, והתשובות בסוף.

**שאלה 1:**

```
{ "name": "דנה לוי", "role": "מנהלת מוצר", "city": "חיפה" }
```

א. Markdown · ב. YAML · ג. JSON · ד. XML

**שאלה 2 — הביטו על הבלוק שבין הקווים (---) למעלה:**

```
---
title: דוח שבועי
tags: [סיכום, פגישה]
---
## עיקרי הדברים
- נקודה ראשונה
```

איזה פורמט הוא הבלוק העליון? א. JSON · ב. MD · ג. XML · ד. YAML

**שאלה 3 — פורמט מתחבא בתוך פורמט:**

```
{ "title": "סיכום", "body": "## החלטות\n- היקף: שלושה מודולים" }
```

מה הפורמט החיצוני, שעוטף הכל? א. JSON · ב. YAML · ג. XML · ד. MD

## התשובות (רק אחרי שעניתם)

- **שאלה 1: ג, JSON.** סוגריים מסולסלים, מירכאות סביב כל מפתח, פסיקים.
- **שאלה 2: ד, YAML.** רווחים בתחילת השורה ונקודתיים, בלי סוגריים. קוראים לבלוק הזה frontmatter, ומה שאחריו הוא MD. שני פורמטים באותו קובץ.
- **שאלה 3: א, JSON עוטף הכל.** בתוך השדה body מתחבא MD. ככה פורמטים נכנסים זה בתוך זה, ולכן שווה לזהות מי עוטף את מי.

(שמתם לב? לא כתבתם אף שורה של אף פורמט. קלוד והסקיל עשו את העבודה, אתם רק זיהיתם.)

## בונוס — XML בפרומפט (החריג ששווה להכיר)

רוצים לנסות את היחיד מהמשפחה שתשתמשו בו בעצמכם? הדביקו לקלוד:

<!-- verbatim (פרומפט מאושר) — לא לתרגם ולא לקצר -->
```
קח את הפרומפט הבא ועטוף כל רכיב שלו בתגית XML מתאימה (הוראה, הקשר, דוגמה),
כדי לתת ל-Claude גבולות חדים בין החלקים. בסוף הסבר בשורה אחת למה זה עוזר.

הנה הפרומפט:
הוראה: סכם את המשוב מהלקוחות לחמישה בולטים.
הקשר: הסיכום מיועד להנהלה הבכירה וצריך להיות תמציתי.
דוגמה: כל בולט עד שתי שורות, ממוקד בפעולה.
```
