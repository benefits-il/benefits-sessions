# סיכום · מסלול ההתקנות

## מה שלמדנו

1. **שיעור 5 בונים — לא מתקינים.** כל ההתקנות נסגרות בבית, כדי שכל דקת-כיתה תלך לבנייה עצמה. את הבדיקות עושה קלוד: מבקשים ממנו לבדוק מה מותקן, והוא מריץ ומדווח.
2. **מה שכבר היה:** ‏Node (מיחידת המסכים) ו-git (משיעור 4) — רק מוודאים.
3. **Python לכולם.** שפת-התשתית שחלק מכלי-הכיתה רצים עליה — מתקינים פעם אחת, לא לומדים לתכנת. הצעד הקריטי במתקין: לסמן **Add python.exe to PATH** לפני Install.
4. **טלגרם בנייד ובמחשב** — בכיתה מקימים סוכן שמדברים איתו שם, ועובדים מהמחשב.
5. **הבוט נולד ב-BotFather.** ‏/newbot ← שם תצוגה ← שם-משתמש שמסתיים ב-bot ← מתקבל **טוקן**. הטוקן הוא אישור למוכ"ז — בדיוק כמו מפתח API — ולכן הוא הולך ישר לכספת, לשדה מוסתר. הבוט עצמו הוא בינתיים טלפון בלי סים: בכיתה מחברים אליו את המוח.

## הצ'קליסט המלא — מגיעים לשיעור 5 עם

מפתח Gemini טעון + כספת (יחידה 1) · ‏NotebookLM MCP מחובר (יחידה 2) · סקיצות-מסכים בתיקייה (יחידה 4) · ‏Node · ‏git · ‏Python · טלגרם · בוט עם טוקן בכספת. את הפריט האחרון — התקנת BMAD — מוסיפים ביחידה הבאה, אחרי סרטון-ההסבר.
