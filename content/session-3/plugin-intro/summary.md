# סיכום · מה זה Plugin

## מה שלמדנו

1. **plugin הוא אריזה, לא מושג חדש.** הוא עוטף יחד כמה דברים שאתם כבר מכירים (סקיל, חיבור, פקודה וסוכן), ומאפשר להתקין את כולם במכה אחת, במקום כל אחד בנפרד.
2. **ארבעה רכיבים מוכרים בפנים.** סקיל הוא יכולת, חיבור הוא דלת לכלי חיצוני, פקודה מפעילה משהו בלחיצה, וסוכן הוא פרסונה עם תפקיד. אין שום רכיב חדש שצריך ללמוד, אלא רק הרכבה שלהם יחד.
3. **מתקינים מתוך Cowork.** עושים את זה דרך Customize ו-Personal plugins, מאותו מקום שמנהל גם את הסקילים והחיבורים. פותחים את ה-plugin ורואים את הרכיבים שבפנים.

## השורה ששווה לזכור

> plugin לא נתן לכם רכיב חדש ללמוד. הוא לקח כמה דברים שכבר ידעתם, וארז אותם לחבילה אחת שמתקינים בלחיצה.

## תזכורת קטנה

ההבחנה ששווה לזכור: plugin הוא האריזה, סקיל הוא יכולת אחת, סוכן הוא פרסונה עם תפקיד, וחיבור הוא דלת לשירות חיצוני. כאן רק התקנו ושיחקנו עם plugin מוכן. את הבנייה של plugin משלכם נעשה בכיתה.
