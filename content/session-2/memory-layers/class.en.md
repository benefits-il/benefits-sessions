# In Class — Memory Layers

## Finish Benefit, and install the two remaining Skills

In class you built Benefit and installed two Skills, save and retrieve. At home you complete the picture: install the two remaining Skills and run each one once.

If you did not get to build Benefit in class, start there. Open the project in Claude, paste the agent instructions from the download page, and connect Notion with read and write permission. If you get stuck, try again, and we will sort it out in the session.

## Where to download
All the Skills and knowledge files sit on one page: **benefits-il.dev/benefit-agent**. That is where you download each Skill and install it in Claude through Settings → Skills → Upload.

## Skill 1 — Advising and breaking down by STUDIO
The Skill that takes a real task of yours and breaks it down by the STUDIO method, with a recommendation for which tool to use at each stage.
- Install it from the download page.
- Run it once on a real task of yours. Paste this prompt to Benefit, and fill in the task in the brackets:
```
I have a task: [your task in one line]. Break it down for me by STUDIO, and tell me which tool to use at each stage.
```

## Skill 2 — Building a prompt by MICRO
The Skill that builds you a clean, organized prompt by the MICRO method.
- Install it from the download page.
- Run it once. Paste this prompt to Benefit, and fill in the brackets:
```
I need a prompt that [what you want the prompt to do]. Build it for me by MICRO.
```

## A few things worth remembering
- The manual approach is always available: you can open Notion and fix a name or a date yourself. It is better to let Benefit do the filing, but when you need to, you can.
- If a write action does not go through, make sure the Notion connection is linked with write permission, and approve the write tool when it asks.
- If you get stuck, try again, and we will sort it out in the session.
