# סיכום · בוחרים עולם

## שלושת העולמות

1. **זומי — הקלטה נכנסת, מסמך-ניתוח יוצא.** בוט-טלגרם עם שני סוכנים, מראיין ומנתח. הזרימה: הקלטה ← שאלון קצר ← תמלול ← מסמך-ניתוח. עובד ישירות מול Gemini, עם המפתח שכבר בכספת. התוצר: בוט שהופך כל הקלטה למסמך.
2. **טוקי — לינק נכנס, פתק-ידע יוצא.** צינור-ידע מסרטונים, כלי-טרמינל ובהמשך בוט. הזרימה: לינק ← קריאת-הסרטון (דיבור + טקסט על המסך) ← פתק-ידע מסווג. עובד ישירות מול Gemini, אותו מפתח. התוצר: ספריית-ידע שגדלה מכל סרטון.
3. **פיקסי — רעיון נכנס, סרטון ממותג יוצא.** סקיל-אווטאר (דמות קבועה) + סוכן-הפקה עם שערי-אישור. הזרימה: רעיון ← דמות ← סצנות ← סרטון אנכי עם המותג שלכם. רוכב על המנוע מיחידה 1 ועל סקיל-המותג. התוצר: אולפן-כיס על המותג שלכם.

## הבחירה

אין תשובה נכונה — בוחרים לפי משיכה, לפי מה שרוצים שיהיה ביד בסוף. עולם שלא נבחר נשאר זמין, ותמיד אפשר לחזור אליו.

## איך בונים

מורידים את הערכה של העולם שנבחר, מחלצים, ופותחים את התיקייה בקלוד קוד. פותחים את START-HERE, מעתיקים את פרומפט-הפתיחה (עברית או אנגלית) ומדביקים לקלוד. מכאן קלוד מוביל — תוכנית אחת בכל פעם, לפי הסדר, עם אישור ביניהן.
