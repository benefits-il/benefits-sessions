# In Class — Foundations

## Rewrite a weak prompt with the five MICRO components

Right after we name MICRO, you take a real meeting from this week and turn a lazy prompt into a full, professional one.

## The instruction

1. Pick one real meeting you had this week. Not an invented example — a meeting that actually happened.
2. Start from this weak prompt:

   > Write me a summary of the meeting.

3. Rewrite it so it carries all five MICRO components. Write at least one full sentence for each component you add:
   - **Mission** — what exactly you want to happen.
   - **Input** — which material to work on: the transcript, the raw notes, the recording.
   - **Context** — the audience, the goal, the background.
   - **Review** — ask the model to explain what it understood and wait for your approval before it continues, not just to stop and ask if something is missing.
   - **Output** — the exact format, length and structure.
4. Read your prompt back and mark which of the five components are really there.

## Desired result

One prompt about a real meeting that holds all five MICRO components — one you could paste into a chat right now and trust the answer it returns.

## Challenge (optional)

Remove one component on purpose, and try to think (or say out loud) which "surprise" it might bring back into the answer you get.
