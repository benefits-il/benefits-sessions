## סיימו את Benefit, והתקינו את שני הסקילים הנותרים

בכיתה בניתם את Benefit והתקנתם שני סקילים, שמירה ושליפה. בבית משלימים את התמונה: מתקינים את שני הסקילים הנותרים ומריצים כל אחד פעם אחת.

אם לא הספקתם לבנות את Benefit בכיתה, התחילו מזה: פתחו את הפרויקט בקלוד, הדביקו את הוראות הסוכן מדף ההורדה, וחברו את Notion עם הרשאת קריאה וכתיבה. אם נתקעתם, נסו שוב, ונסדר את זה במפגש.

## איפה מורידים
כל הסקילים וקבצי הידע יושבים בעמוד אחד: **benefits-il.dev/benefit-agent**. שם מורידים כל סקיל ומתקינים אותו בקלוד דרך Settings ← Skills ← Upload.

## סקיל 1 — ייעוץ ופירוק לפי STUDIO
הסקיל שלוקח משימה אמיתית שלכם ומפרק אותה לפי שיטת STUDIO, עם המלצה באיזה כלי להשתמש בכל שלב.
- התקינו אותו מדף ההורדה.
- הריצו אותו פעם אחת על משימה אמיתית שלכם. הדביקו ל-Benefit את הפרומפט הבא, והשלימו את המשימה בסוגריים:
```
יש לי משימה: [המשימה שלכם בשורה אחת]. תפרק לי אותה לפי STUDIO, ותגיד באיזה כלי כדאי להשתמש בכל שלב.
```

## סקיל 2 — בונה פרומפט לפי MICRO
הסקיל שבונה לכם פרומפט נקי ומסודר לפי שיטת MICRO.
- התקינו אותו מדף ההורדה.
- הריצו אותו פעם אחת. הדביקו ל-Benefit את הפרומפט הבא, והשלימו בסוגריים:
```
אני צריך פרומפט ש[מה שאתם רוצים שהפרומפט יעשה]. תבנה לי אותו לפי MICRO.
```

## כמה דברים שכדאי לזכור
- גישה ידנית תמיד זמינה: אפשר לפתוח את Notion ולתקן שם או תאריך בעצמכם. עדיף לתת ל-Benefit לתייק, אבל כשצריך, אתם יכולים.
- אם פעולת כתיבה לא עוברת, ודאו שחיבור Notion מחובר עם הרשאת כתיבה, ואשרו את כלי הכתיבה כשהוא מבקש.
- אם נתקעתם, נסו שוב, ונסדר את זה במפגש.
