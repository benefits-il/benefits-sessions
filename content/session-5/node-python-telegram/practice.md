# בבית — מסלול ההתקנות

## מה עושים כאן

שיעור 5 הוא שיעור של בנייה אמיתית: תבנו מנוע-מדיה, תקימו סוכן חי, ותתחילו את הפרויקט הסופי. כדי שכל זה יקרה — אף דקת-כיתה לא נשרפת על התקנות. הכול מוכן מהבית.

החדשות הטובות: רוב הדברים כבר אצלכם. ביחידה הזאת מוודאים את מה שיש, מתקינים שני דברים חדשים (Python וטלגרם), ויוצרים את הדבר הכי כיפי — **בוט טלגרם משלכם**, שבכיתה יקבל מוח.

> **נתקעתם? שאלו את קלוד.** צלמו מסך, הדביקו לקלוד, ובקשו: "מה אני רואה כאן ואיך אני ממשיך?".

## צעד 1 — מוודאים את מה שכבר יש

שני דברים אמורים להיות מותקנים אצלכם מהדרך:
- **Node** — התקנתם ביחידת המסכים (ההכנה הקודמת).
- **git** — מותקן ומחובר משיעור 4.

איך בודקים? הכי פשוט — שואלים את קלוד. פותחים את Claude Code ומדביקים:

```text
בדוק בבקשה אם Node ו-git מותקנים אצלי, ותגיד לי מה הגרסאות
```

או, אותו פרומפט באנגלית:

```text
Please check whether Node and git are installed on my machine, and tell me the versions
```

קלוד יריץ את הבדיקות ויחזיר תשובה. משהו חסר? קלוד גם יסביר איך להשלים (התקנת Node מחכה בצעד 1 של יחידת המסכים).

## צעד 2 — מתקינים Python

‏Python היא שפת-התשתית שחלק מהכלים שנבנה בכיתה רצים עליה. לא לומדים לתכנת בה — רק מתקינים אותה פעם אחת, כמו Node.

מה עושים:

1. נכנסים ל-python.org, עוברים ל-Downloads, ומורידים את הגרסה המוצעת ל-Windows.
2. מריצים את המתקין — ו**כאן הצעד החשוב באמת:** במסך הראשון מסמנים את התיבה **Add python.exe to PATH** לפני שלוחצים Install. בלי הסימון הזה, המחשב לא ימצא את Python ושום דבר לא ירוץ.
3. ‏Install ← מחכים ← ‏Close.
4. בודקים עם קלוד:

```text
בדוק בבקשה אם Python מותקן אצלי ומה הגרסה
```

או, אותו פרומפט באנגלית:

```text
Please check whether Python is installed on my machine and what version
```

מה השגתם: ‏Python מותקן וזמין — התשתית לכלים של הכיתה.

בדיקה עצמית:
- קלוד מאשר ש-Python מותקן ומחזיר מספר גרסה.

## צעד 3 — פותחים חשבון טלגרם

בכיתה תקימו סוכן שמדברים איתו בטלגרם — אז צריך טלגרם.

1. מתקינים את טלגרם בנייד (מחנות האפליקציות) ופותחים חשבון עם מספר הטלפון.
2. מתקינים את טלגרם **גם על המחשב** (desktop.telegram.org) ומתחברים עם אותו חשבון — בכיתה נעבוד מהמחשב, וזה הרבה יותר נוח.

כבר יש לכם טלגרם? מדלגים ישר לצעד הבא.

## צעד 4 — יוצרים את הבוט שלכם (הרגע הכיפי)

עכשיו יוצרים בוט אמיתי משלכם. בטלגרם יש "בוט שמייצר בוטים" — קוראים לו BotFather, והוא הדרך הרשמית. הבוט שתיצרו עכשיו הוא כמו טלפון בלי סים: הוא קיים, יש לו שם — ובכיתה נחבר אליו את המוח.

מה עושים:

1. בטלגרם מחפשים **BotFather** ופותחים איתו שיחה (זה החשבון עם וי כחול של אימות).
2. שולחים לו:

```text
/newbot
```

3. הוא ישאל לשם הבוט — נותנים שם תצוגה (מותר בעברית, למשל "הסוכן של דנה").
4. הוא יבקש שם-משתמש ייחודי — באנגלית, בלי רווחים, **חייב להסתיים ב-bot** (למשל dana_agent_bot). אם השם תפוס, מוסיפים מספר או משנים קצת.
5. וברגע שהבוט נוצר — ‏BotFather שולח לכם **טוקן**: שורה ארוכה של אותיות ומספרים. עצרו כאן רגע.

**הטוקן הזה הוא מפתח לכל דבר.** מי שמחזיק בו שולט בבוט שלכם — בדיוק כמו שלמדתם ביחידה הראשונה: אישור למוכ"ז. אז עושים איתו בדיוק מה שעשיתם עם מפתח ה-Gemini:

6. פותחים את Bitwarden ובונים רשומה כמו שהתאמנתם: פריט Note, שם ברור (למשל Telegram Bot Token), והטוקן בשדה מוסתר. לא משאירים אותו בצ'אט, לא שומרים בקובץ על שולחן-העבודה.

מה השגתם: בוט טלגרם רשום על שמכם, והטוקן שלו שמור בכספת — מוכן לקבל מוח בכיתה.

בדיקה עצמית:
- הבוט מופיע אצלכם בטלגרם (חפשו את שם-המשתמש שבחרתם).
- הטוקן בכספת, בשדה מוסתר.
